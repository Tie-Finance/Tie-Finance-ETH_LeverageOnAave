
## How to test with truffle

### node version require
 v20.16.0

### install ganache
 npm install ganache --global
 
### run ganache
 ganache 

### install package
 in the folder ./Tie-Finance-ETH_LeverageOnAave run the command:
 
 npm install  
 
### run test
 in the folder ./Tie-Finance-ETH_LeverageOnAave run the command:
 
 truffle test
